Q1. What does the replace() method do in Python? How many arguments does it take?
Ans:-
The replace() method replaces a specified word or character with another word or character in a string.

Syntax:

string.replace(old, new, count)
old → text to be replaced
new → replacement text
count → optional; number of replacements

It mainly takes 2 required arguments and 1 optional argument.

Example:

text = "I like tea"
print(text.replace("tea", "coffee"))

Output:

I like coffee


Q2. What is the difference between lstrip(), rstrip(), and strip()? Give one example of each.
Ans:-
These methods remove whitespace (or specified characters).

Method	Removes From
lstrip()	Left side
rstrip()	Right side
strip()	Both sides

Examples:

text = "   Python   "

print(text.lstrip())
print(text.rstrip())
print(text.strip())

Output:

"Python   "
"   Python"
"Python"


Q3. What does split() return? What is the default delimiter if none is provided?
Ans:-
split() breaks a string into parts and returns a list.

If no delimiter is given, the default delimiter is whitespace (space).

Example:

text = "Python is easy"
print(text.split())

Output:

['Python', 'is', 'easy']

Q5. What does swapcase() do? What will "Hello World".swapcase() return?
Ans:-
swapcase() changes:

uppercase → lowercase
lowercase → uppercase

Example:

print("Hello World".swapcase())

Output:

hELLO wORLD

Q8. What does join() do? Which object calls join() -- the list or the separator string?
Ans:-
join() combines elements of a list into a single string.

The separator string calls join(), not the list.

Example:

words = ["Python", "is", "fun"]

print(" ".join(words))

Output:

Python is fun


Q9. What is the difference between split() and partition()?
Ans:-

        Method	                          Return Type	                                Behavior
        split()	                             List	                              Splits all occurrences
        partition()	                         Tuple	                              Splits only first occurrence

Example:

text = "apple-banana-orange"

print(text.split("-"))
print(text.partition("-"))

Output:

['apple', 'banana', 'orange']
('apple', '-', 'banana-orange')


Q10. What does strip() remove from a string? Does it remove whitespace from the middle of a string?
Ans:-
strip() removes whitespace from the beginning and end of a string.

It does NOT remove spaces from the middle.

Example:

text = "   Hello World   "

print(text.strip())

Output:

Hello World

Q19. What will "hello".upper().swapcase() return? Explain method chaining with this example.
Ans:-
print("hello".upper().swapcase())

Output:

hello

Explanation:

"hello".upper() → "HELLO"
"HELLO".swapcase() → "hello"

This is called method chaining, where one method is called after another.
