Q1. What is Console I/O in Python? What is the difference between input and output?
Ans:-
Console I/O means taking data from the keyboard and displaying data on the screen.

Input → Data entered by the user using input()
Output → Data shown to the user using print()

Example:

name = input("Enter your name: ")
print(name)

Q2. What does the print() function do? Write its full syntax including sep and end parameters.
Ans:-
    print() function used to display the data 
    syntax : 
            (1) print("Shankar Iragond ",sep="-",end="\n")
            (2) name="Shankar"
                surname="Iragond"
                print("Name : {},Surname :{}".format(name,surname))
            (3) print(f"Name : {name}, Surname : {surname}")

sep → Separator between values
end → What to print at the end
