1.What is an identifier in Python?
Ans:-
An identifier is the name used to identify variables, functions, classes, modules, etc., in Python.


2.What are the rules for creating identifiers in Python?
Ans:-
Must start with a letter (A–Z or a–z) or underscore (_)
Cannot start with a number
Can contain letters, digits, and underscores
Cannot contain spaces or special characters like @, #, $, etc.
Cannot use Python keywords
Identifiers are case sensitive


3.Can we use keywords as variable names? Why?
Ans:-
No. Keywords have special meanings in Python, so they cannot be used as variable names.


4.What is meant by Python being case sensitive?
Ans:-
Python treats uppercase and lowercase letters as different characters.


5.What is the difference between A and a in Python?
Ans:-
A and a are treated as two different identifiers because Python is case sensitive.


6.What are Python keywords?
Ans:-
Keywords are reserved words in Python that have predefined meanings and special purposes.


7.Why can’t we use for, if, or class as variable names?
Ans:-
Because they are Python keywords reserved for specific programming functions.
